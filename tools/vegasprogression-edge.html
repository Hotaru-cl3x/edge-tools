Vegas Progression Edge

ベガスプログレッション法をリアルタイム自動計算するツール（vegasprogression-tool.js）。2→4→6→9→14→21単位の6ステップ固定ラダーをSTEPS配列で実装。初期ベットが2単位。6連勝で自動サイクルリセット。2連勝後からprofit-lock状態に入る。

ツールを無料で使う →

基本仕様

システム種別Positive Progression / Fixed 6-step
対象ゲームバカラ・ルーレット（2倍配当）
ステップ配列STEPS = [2, 4, 6, 9, 14, 21]
初期ベット2単位（他システムより高い）
リスクMedium

実装アルゴリズム

JavaScriptソースコードから抽出したコアロジックです。

// 6ステップラダー（vegasprogression-tool.js）
var STEPS = [2, 4, 6, 9, 14, 21];
S = { si:0, ... }

if (r === 'win') {
    S.pnl += ba; S.si++;
    if (S.si >= STEPS.length) {
        S.si = 0;  // 6連勝 → 自動リセット
        S.hist.unshift({ ..., complete: true });
        draw(); return;
    }
} else {
    S.pnl -= ba; S.si = 0;  // 2単位にリセット
}

// profit-lock表示（si >= 2、つまり2連勝後から）
if (S.si >= 2) {
    var safeAmt = S.pnl - ba;  // 今負けても残る利益
    if (safeAmt >= 0) {
        // "今負けても損失なし！確保済み利益: +X" を表示
    }
}

数式・計算仕様

ステップ配列STEPS = [2, 4, 6, 9, 14, 21]
profit-lock開始si >= 2（2連勝後、3回目ベット=6単位）
6連勝時の総獲得2+4+6+9+14+21 = 56単位
1回の失敗コスト2単位（他システムより高い）
safeAmt計算pnl - nextBet（次ベット差し引き後の確定利益）

メリット・デメリット

✓ メリット

  • 6ステップで連勝ボーナスが大きい（最大21単位）
  • si >= 2でprofit-lockが早期に始まる
  • サイクル完了履歴でトロフィー表示

✗ デメリット・注意点

  • 初期ベットが2単位で失敗コストが高い
  • 6連勝の達成確率は(0.5)^6 ≈ 1.6%
