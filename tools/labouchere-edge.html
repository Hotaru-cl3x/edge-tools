Labouchere Edge

ラブシェール法をリアルタイム自動計算するツール。目標利益額から数列を自動生成するsplitSequence関数を内蔵。リスク設定（small/medium/large）で数列の分割方式を切り替えるアルゴリズムを搭載。

基本仕様

システム種別Sequence-based / Negative Progression
対象ゲームバカラ・ルーレット（2倍配当）
数列生成目標利益額から自動生成
リスク設定small / medium（推奨）/ large の3段階
複雑度Medium

実装アルゴリズム

JavaScriptソースコードから抽出したコアロジックです。

// 数列自動生成（labouchere-tool.js）
function splitSequence(target, risk) {
    var units = Math.round(target);  // 目標利益（単位数）
    if (risk === 'small') {
        // 全て1単位に分割（最長・低リスク）
        var arr = []; for (var i=0; i<units; i++) arr.push(1); return arr;
    } else if (risk === 'large') {
        // 2分割: [floor(0.4n), ceil(0.6n)]（最短・高リスク）
        var a = Math.floor(units * 0.4); return [a, units - a];
    } else {  // medium
        // 3〜6分割（count = min(6, max(3, floor(n/2)))）
        var count = Math.min(6, Math.max(3, Math.floor(units / 2)));
        // remainingを均等に分配し末尾に残余を追加
    }
}

数式・計算仕様

目標達成条件seq.length === 0（全要素の消去）
small設定の数列長目標単位数と同じ（全て1）
large設定の数列長常に2要素（[0.4n, 0.6n]）
medium設定の数列長min(6, max(3, floor(n/2))) 要素
初回ベット（medium）seq[0] + seq[末尾]

メリット・デメリット

✓ メリット

  • 目標利益額から数列を自動生成するアルゴリズム
  • 3段階のリスク設定で戦略を柔軟に調整

  • 連敗で数列が長くなりベット額が増大
  • 目標が高いほど数列総和が大きくリスクも増す
