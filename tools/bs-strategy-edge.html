BS Strategy Edge

ビデオブラックジャック専用のベーシックストラテジー参照ツール。ディーラーのアップカードとプレイヤーの手札を選択するだけで最適アクション（HIT/STAND/DOUBLE/SPLIT）を即座に表示。バスト率も同時計算。外部JSファイルなしのHTML単体動作。

ツール種別Decision Matrix / Basic Strategy Reference

実装アルゴリズム

JavaScriptソースコードから抽出したコアロジックです。

// ストラテジーテーブル（HTMLインライン実装）
// 外部JSファイルなし（bs-strategy-edgeのコピー.htmlに内蔵）

// ① ストラテジーテーブル（2次元マップ）
// キー: [playerHand][dealerUpcard]
const strategy = {
    'hard9':  { 2:'H', 3:'D', 4:'D', 5:'D', 6:'D', 7:'H', ... },
    'hard10': { 2:'D', 3:'D', 4:'D', 5:'D', 6:'D', 7:'D', 8:'D', 9:'D', 10:'H', A:'H' },
    'hard11': { 2:'D', 3:'D', 4:'D', 5:'D', 6:'D', 7:'D', 8:'D', 9:'D', 10:'D', A:'H' },
    // Soft totals, Pairs等も同様に定義
    // H=Hit, S=Stand, D=Double, SP=Split
};

// ② バスト確率計算
function calcBustProbability(handTotal) {
    var bustCount = 0;
    for (var card = 1; card <= 13; card++) {
        var value = Math.min(card, 10);  // 10以上は10扱い
        if (handTotal + value > 21) bustCount++;
    }
    return (bustCount / 13 * 100).toFixed(1) + '%';
}
// カード分布: 2-9各1/13, 10/J/Q/K合計4/13, A=1/13

数式・計算仕様

テーブルの次元プレイヤー手札 × ディーラーアップカード
バスト確率bustCount / 13（均等分布の仮定）
10枚カードの重み4/13（10, J, Q, K）
Aの扱い1 or 11（ソフトハンドの場合は11として計算）
判定根拠ビデオBJの固定ルールに最適化されたEV計算結果

メリット・デメリット

✓ メリット

  • 外部JS不要でHTML単体動作（ロードが軽い）
  • ストラテジーテーブルをマップで実装し変更に対応しやすい
  • バスト確率の即時表示でリスクを数値で把握できる

✗ デメリット・注意点

  • ビデオBJの固定ルール専用（ライブBJやカスタムルールには非対応）
  • カードカウンティングやデッキ組成の変動を考慮しない
